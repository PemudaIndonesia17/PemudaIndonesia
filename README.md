# PemudaIndonesia
Emotion Calendar adalah sebuah aplikasi web sederhana yang dirancang untuk membantu pengguna mencatat dan memantau suasana hati mereka setiap hari melalui tampilan kalender interaktif. Pengguna dapat memilih emoji emosi sesuai perasaan pada hari tertentu, sehingga di akhir bulan dapat melihat pola mood dan refleksi perkembangan emosi mereka.
